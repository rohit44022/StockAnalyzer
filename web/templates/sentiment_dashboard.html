Social Media Sentiment

Real-time sentiment analysis from Google News, Reddit, Indian Financial RSS, StockTwits, X.com, and NewsAPI.

Collects posts & articles about a stock → analyzes with NLP (VADER + financial lexicon) → tells you what social media thinks

📡 How does this work?
This system collects the latest news articles and social media posts about a stock from 4 different sources. Each post is scored using VADER sentiment analysis (a proven NLP model for social media) augmented with a custom financial lexicon of 100+ stock-market terms (e.g., "breakout", "multibagger", "upper circuit", "promoter selling").
Sources are weighted by reliability and aggregated into a single verdict with confidence level.
⚠️ Social sentiment is ONE input — never trade based on sentiment alone. Always combine with technical and fundamental analysis.

Data Collection
Posts are collected from multiple sources in parallel:
Google News RSS (25% weight) — free, reliable, excellent Indian stock coverage.
Reddit (20% weight) — r/IndianStockMarket, r/IndianStreetBets, etc.
Indian Financial RSS (20% weight) — MoneyControl, ET Markets, LiveMint, Biz Standard.
StockTwits (15% weight) — free, stock-specific social platform where traders tag posts as Bullish/Bearish.
X / Twitter (15% weight) — optional, needs Bearer Token in .env ($100+/month API).
NewsAPI (15% weight) — optional, needs API key in .env file.
NLP Scoring
Each post is scored using VADER (Valence Aware Dictionary and sEntiment Reasoner) — a battle-tested NLP model for social media sentiment. VADER handles capitalization, punctuation, emoji, negation, and degree modifiers.

We augment VADER with a custom financial lexicon of 100+ stock-market terms so it understands phrases like "multibagger", "upper circuit", "death cross", and "promoter pledge".

Confidence = f(post count, source agreement, score variance). More posts + more agreement = higher confidence.
